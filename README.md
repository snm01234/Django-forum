# Django-forum
django로 개발한 웹 게시판 / 박응용님의 pybo 게시판 클론코딩입니다.


구현기능:

* 로그인&로그아웃

* 회원가입

* 글쓴이 표시

* 게시물 수정&삭제 

* 질문, 답변 글에 대한 댓글 작성 

* 게시물 추천 기능

* 마크다운 기능

* 게시물 검색 기능

* 최신순, 추천순, 인기순 정렬 기능

* 조회수, 게시날짜 확인
